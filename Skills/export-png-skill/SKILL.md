# export png

## When to use this skill
Use this skill when the user needs to export png from the code diagram file

## How to extract text
1. Access to files of diagrams code
2. Export png

## Where is source
1. The path to files of diagrams:

  - Sequence diagram: ./Diagrams/UCxx/SDxx.txt

2. Where to store output file
- Activity diagram: ./DiagramsPNG/ActivityDiagrams/
- Sequence diagram: ./DiagramsPNG/SequenceDiagrams/
