@startuml
title UC-39: Ẩn đánh giá vi phạm

actor "Nhân viên Sales" as Staff
boundary ReviewManagement_UI as UI
control Review_Controller as Ctrl
database DB

activate Staff
activate UI
activate Ctrl
activate DB

Staff -> UI : 1. Nhấn nút "Ẩn đánh giá"
UI -> Staff : 2. Hiển thị hộp thoại yêu cầu nhập lý do

Staff -> UI : 3. Nhập lý do và nhấn xác nhận
UI -> Ctrl : 4. Yêu cầu ẩn đánh giá(reviewID, reason)

alt x Nhân viên đã nhập lý do giải trình
    Ctrl -> DB : 5. hideReview(reviewID)
    DB -> Ctrl : 6. success
    Ctrl -> DB : 7. writeAuditLog(staffID, "Hide Review", reason)
    DB -> Ctrl : 8. success
    Ctrl -> UI : 9. Thông báo ẩn đánh giá thành công
    UI -> Staff : 10. Hiển thị thông báo thành công
else x Nhân viên để trống lý do
    Ctrl -> UI : 11. Báo lỗi yêu cầu nhập lý do theo quy định
    UI -> Staff : 12. Hiển thị cảnh báo bắt buộc điền lý do
end

deactivate Staff
deactivate UI
deactivate Ctrl
deactivate DB
@enduml
