@startuml
title UC-37: Xem phản hồi mới nhất

actor "Nhân viên Sales" as Staff
boundary ReviewManagement_UI as UI
control Review_Controller as Ctrl
database DB

activate Staff
activate UI
activate Ctrl
activate DB

Staff -> UI : 1. Chọn mục Quản lý đánh giá
UI -> Ctrl : 2. Yêu cầu danh sách phản hồi mới nhất()
Ctrl -> DB : 3. getLatestReviews()

alt x Hệ thống có dữ liệu đánh giá
    DB -> Ctrl : 4. danh sách ReviewData
    Ctrl -> UI : 5. Trả về dữ liệu phản hồi (tên khách, sản phẩm, số sao...)
    UI -> Staff : 6. Hiển thị danh sách đánh giá lên màn hình
    
    opt x Nhân viên lọc theo số sao
        Staff -> UI : 7. Chọn bộ lọc (1-2 sao)
        UI -> Ctrl : 8. Yêu cầu lọc đánh giá(starRating)
        Ctrl -> UI : 9. Cập nhật danh sách hiển thị
    end
    
else x Chưa có bài đánh giá nào
    DB -> Ctrl : 10. null / empty
    Ctrl -> UI : 11. Thông báo hệ thống chưa có phản hồi
    UI -> Staff : 12. Hiển thị thông báo danh sách trống
end

deactivate Staff
deactivate UI
deactivate Ctrl
deactivate DB
@enduml
