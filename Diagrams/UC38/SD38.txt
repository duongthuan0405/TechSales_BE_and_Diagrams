@startuml
title UC-38: Phản hồi đánh giá

actor "Nhân viên Sales" as Staff
boundary ReviewManagement_UI as UI
control Review_Controller as Ctrl
database DB


activate Staff
activate UI
Staff -> UI : 1. Chọn phản hồi và nhấn nút "Trả lời"
UI -> Staff : 2. Hiển thị khung nhập văn bản

Staff -> UI : 3. Nhập nội dung phản hồi và nhấn gửi
activate Ctrl
UI -> Ctrl : 4. Yêu cầu gửi phản hồi(reviewID, content)

alt x Nội dung phản hồi hợp lệ
    activate DB
    Ctrl -> DB : 5. saveResponse(reviewID, staffID, content)
    DB -> Ctrl : 6. success
    deactivate DB
    Ctrl -> UI : 7. Thông báo gửi phản hồi thành công
    UI -> Staff : 8. Hiển thị thông báo thành công
else x Nội dung phản hồi để trống
    Ctrl -> UI : 9. Báo lỗi nội dung không được để trống
    deactivate Ctrl
    UI -> Staff : 10. Hiển thị cảnh báo yêu cầu nhập liệu
    deactivate Staff
    deactivate UI
end

@enduml
