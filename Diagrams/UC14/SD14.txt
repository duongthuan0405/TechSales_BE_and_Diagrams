@startuml
skinparam sequenceMessageAlign center
skinparam monochrome true
title UC-14: Thêm sản phẩm vào giỏ hàng

actor "Khách hàng" as User
boundary "ProductDetail_UI" as UI
control "Cart_Controller" as Ctrl
database "DB" as DB

User -> UI: 1. Nhấn nút thêm sản phẩm vào giỏ hàng
activate User
activate UI

UI -> Ctrl: 2. Gửi yêu cầu thêm sản phẩm với số lượng cụ thể
activate Ctrl

Ctrl -> DB: 3. checkInventory(productID)
activate DB

DB --> Ctrl: 4. Phản hồi số lượng tồn kho hiện có

alt [x Số lượng yêu cầu vượt quá tồn kho]
    Ctrl --> UI: 5. Phản hồi không đủ hàng và điều chỉnh số lượng
    UI --> User: 6. Hiển thị cảnh báo và cập nhật lại ô nhập liệu
else [x Số lượng hợp lệ]
    Ctrl -> DB: 7. addProductToCart(userID, productID, qty)
    DB --> Ctrl: 8. Xác nhận lưu dữ liệu thành công
    
    Ctrl --> UI: 9. Phản hồi kết quả thành công
    UI --> User: 10. Hiển thị thông báo thêm vào giỏ hàng thành công
end

deactivate DB
deactivate Ctrl
deactivate UI
deactivate User
@enduml
